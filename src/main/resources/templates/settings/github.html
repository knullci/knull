How to set up GitHub Integration:

  1. Go to GitHub → Settings → Developer settings → Personal access tokens
  2. Generate a new token (classic) with the following permissions:
    • repo - Full control of private repositories
    • repo:status - Access commit status
  3. Copy the generated token
  4. Go to Credentials and create a new Token credential with your GitHub token
  5. Return here and select the credential from the dropdown above
  6. Click "Save Configuration"
