GitHub Integration

Select a credential that contains your GitHub Personal Access Token

How to set up:

  1. Create a GitHub Personal Access Token with repo and status permissions
  2. Go to Credentials and create a new Token credential
  3. Select the credential above and save
  4. Your builds will now use this token to update commit statuses on GitHub
